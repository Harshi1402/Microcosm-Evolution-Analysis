# Data Structure (No Raw Data Included)

Raw data is not included for ethical reasons.

## Expected Columns

- pop — population ID (1–10)
- ft1–ft5 — functional traits
- mt1–mt5 — morphological traits
- sct1–sct3 — semi‑cryptic traits
- gm1–gm20 — neutral genetic markers (0, 1, or 2 copies)

## File Naming Convention
all e1 genX.csv  
Where X = generation number (1–2500)

Scripts in `/scripts/` assume this structure.
